<b>Google Drive Converter</b> is a basic converter from files stored in Google Drive to direct links to them.  
Without clicking any button!<br/>
<img width="125px" alt="sad button" src="https://i.imgur.com/HdQKrzN.png"/>

## Objective:

:books: To learn React Bootstrap & Router and to help people share files more easily.

## Currently supports:

<p float="center">
<img alt="navbar" src="https://i.imgur.com/euk4sZt.png"/ style="width:100%">
</p>

- Native Google files:
  - Sheets
  - Docs
  - Slides
  - Drawings
- Reader mode (PDFs, embeds, etc)
- With File ID (Video, audio)

## Security:

All the processes to transform links are done in the browser and are not stored anywhere else but your clipboard,  
if you still feel a bit skeptical, you can check the code yourself :wink:
